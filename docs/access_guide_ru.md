# UNET: быстрая инструкция

Короткая инструкция для телефона и ПК. Профиль доступа вы получаете в боте @UNET_shop_bot.

## Телефон (iOS / Android)

1. Установи совместимое приложение:
   - iOS: Streisand / Shadowrocket
   - Android: v2rayNG / NekoBox
2. Открой бота **@UNET_shop_bot** и выбери тариф.
3. Получи профиль доступа для совместимого приложения.
4. Добавь профиль доступа в приложение и подключись.

## ПК (Windows / macOS)

1. Установи совместимое приложение:
   - Windows: v2rayN
   - macOS: Streisand (или другой совместимый клиент)
2. Получи профиль доступа в боте **@UNET_shop_bot**.
3. Добавь профиль доступа в приложение и подключись.

## Если не получилось

Открой поддержку в боте — поможем выбрать приложение и проверить настройки.
